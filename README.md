## 🚀 Getting Started

### Prerequisites

- Python 3.8+
- PyTorch
- Jupyter Notebook or Google Colab (recommended)

### Install dependencies

```bash
pip install torch jupyter
```

### Run locally

```bash
git clone https://github.com/prateek-anand/Experimenting-with-AI.git
cd Experimenting-with-AI
jupyter notebook
```

Or open any `.ipynb` file directly in [Google Colab](https://colab.research.google.com/) — that's where these were originally developed (tested on Tesla T4 GPU).
